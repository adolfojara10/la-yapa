# Yapi mascot — PLACEHOLDER assets

These SVGs are intentionally minimal stand-ins so screens render correctly
during development. They are NOT final brand artwork.

## States included

| File                        | Use case                                      |
| --------------------------- | --------------------------------------------- |
| `yapi-base.svg`             | Neutral / fallback                            |
| `yapi-happy.svg`            | Onboarding, success states                    |
| `yapi-sleepy.svg`           | Empty states (no bags nearby, no orders yet)  |
| `yapi-celebrating.svg`      | Achievement unlocks, first purchase, streaks  |
| `yapi-sad.svg`              | Errors, expired bags, cancellation flows      |
| `yapi-chef.svg`             | Restaurant / business onboarding              |
| `yapi-eco-warrior.svg`      | Impact dashboards, sustainability cards       |
| `yapi-with-bag.svg`         | Bag listing empty state, checkout success     |
