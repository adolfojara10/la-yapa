# La Yapa — Logo (PLACEHOLDER)

These are stand-in SVGs so the brand is visible during development. The final
artwork should be commissioned per the master vision brief.

## Files

| File                  | Variant                          |
| --------------------- | -------------------------------- |
| `logo-color.svg`      | Full color + tagline             |
| `logo-mono-dark.svg`  | Mark-on-cream + dark wordmark    |
| `logo-mono-white.svg` | All-white (for dark backgrounds) |
| `logo-mark.svg`       | Mark only (no wordmark) — 1:1    |

Mirrored to `apps/admin/public/logo/` so Next.js can serve them at
`/logo/logo-color.svg` etc.
