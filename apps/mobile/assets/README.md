# Mobile assets

Drop your Expo assets here before first build:

- `icon.png` — 1024×1024 PNG (app icon)
- `splash.png` — 2048×2048 PNG (splash screen)
- `adaptive-icon.png` — 1024×1024 PNG (Android adaptive icon foreground)
- `favicon.png` — 48×48 PNG (web favicon)

These are referenced from `app.json`. Until they exist Expo will warn but still run.
